Django Couch
============


Requirements
------------

`Django <https://www.djangoproject.com/>`_ 1.11 or later


Installation
------------

::

    $ pip install django-couch


Setup
-----

Add ``'couch'`` to INSTALLED_APPS in your settings.py::

    INSTALLED_APPS = (
        # ...

        'couch',

        # ...
    )
